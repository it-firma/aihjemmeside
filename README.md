# aihjemmeside.no

Komplett norsk guide som viser hvordan du lager en hjemmeside med AI. Bygget med Domeneshop, ChatGPT, Claude, GitHub og Vercel for 99 kroner på 47 minutter.

## Live

https://aihjemmeside.no

## Struktur

```
aihjemmeside-komplett/
├── index.html                     Hovedsiden
├── 404.html                       Egendefinert 404 side
├── llms.txt                       LLM discovery med alle priser
├── robots.txt                     Crawler instruksjoner
├── sitemap.xml                    For søkemotorer
├── site.webmanifest               PWA manifest
├── entity-index.json              Strukturerte data for AEO
├── vercel.json                    Vercel konfigurasjon
├── .gitignore                     Git ignorer filer
├── prompt-mal/index.html          Prompt mal
├── domene-guide/index.html        Domene guide
├── claude-vs-chatgpt/index.html   AI sammenligning
└── publisering/index.html              Publiseringsguide
```

## Teknologi

Ren HTML, CSS og vanilla JavaScript. Ingen build steg, ingen npm, ingen rammeverk. Alt inline i hver HTML fil.

## Deployment

1. Opprett privat GitHub repository
2. Last opp filene
3. Importer til Vercel
4. Legg til eget domene under Settings → Domains
5. Oppdater DNS hos domeneregistrar
